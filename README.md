Task 02
• Целевой файл: src/components/WelcomeMessage.jsx
• Действие: Реализуйте паттерн "Ранний возврат" (Early Return).
1. Примите пропc `isAuth` (boolean).
2. Если `isAuth` равен false, верните `null` (или пустой фрагмент), чтобы компонент ничего не рендерил.
3. Если `isAuth` равен true, верните заголовок `<h2>Добро пожаловать, Пользователь!</h2>`.
• CSS-подсказка: Не требуется.
• Проверка: Код: Используется конструкция `if (!isAuth) return ...`.

Task 03
• Целевой файл: src/components/Greeting.jsx
• Действие: Реализуйте паттерн "Тернарный оператор".
1. Примите пропс `isLoggedIn` (boolean).
2. Внутри `return` используйте тернарный оператор.
3. Если `isLoggedIn` true — покажите `<h3>С возвращением!</h3>`.
4. Иначе — покажите `<h3>Пожалуйста, войдите в систему.</h3>`.
• CSS-подсказка: Не требуется.
• Проверка: Код: Использован синтаксис `condition ? true : false`.

Task 04
• Целевой файл: src/components/Notification.jsx
• Действие: Реализуйте паттерн "Логическое И" (Logical AND).
1. Примите пропс `messages` (массив строк).
2. Используйте оператор `&&` для рендеринга блока `div`, только если длина массива больше 0.
3. Внутри блока выведите: `У вас {messages.length} новых сообщений`.
4. ВАЖНО: Убедитесь, что вы не рендерите "0" при пустом массиве. Используйте строгое сравнение длины (`length > 0`).
• CSS-подсказка: Не требуется.
• Проверка: Визуально: При пустом массиве на экране пусто, цифра "0" не отображается.

Task 05
• Целевой файл: src/components/UserProfile.jsx
• Действие: Реализуйте комбинированный подход.
1. Примите пропс `user` (объект или null).
2. Если `user` не передан (null) — верните `<div className="warning">Профиль не найден</div>` (используйте тернарный оператор или if).
3. Если `user` существует, верните `div` с именем пользователя (`user.name`).
4. Внутри этого же `div`, используя оператор `&&`, выведите возраст (`user.age`), но только если возраст больше 18.
• CSS-подсказка: Не требуется.
• Проверка: Код: Присутствует проверка на существование объекта `user` перед обращением к его свойствам.

Task 06
• Целевой файл: src/components/TodoList.jsx
• Действие: Реализуйте условный рендеринг стилей и тегов.
1. Примите пропсы `task` (строка) и `isCompleted` (boolean).

• Проверка: Визуально: Выполненная задача перечеркнута.
