РАЗДЕЛ 2: ЗАДАЧИ

Task 01
• Целевой файл: src/App.css
• Действие: Реализуйте глобальные стили для "Sticky Footer" (прижатого подвала) и сброс отступов.
• CSS-подсказка:
```css
* { margin: 0; padding: 0; box-sizing: border-box; }
html, body { height: 100%; }
#root { display: flex; flex-direction: column; min-height: 100vh; }
main { flex: 1; }
img { max-width: 100%; }
```
• Проверка: Визуально: если контента на странице мало, подвал все равно находится строго внизу окна браузера.

Task 02
• Целевой файл: src/components/Header/Header.jsx
• Действие: Создайте шапку сайта, используя **инлайн-стили** (inline styles).
1. Внутри компонента (перед `return`) создайте объект стилей (например, `headerStyles`). Задайте в нем: цвет фона (любой светлый), паддинги и выравнивание текста по центру.
2. Примените этот объект к тегу `<header>` через атрибут `style`.
3. Внутри header добавьте заголовок `h1` с названием урока.
• CSS-подсказка: Не требуется (используем JS-объект).
• Проверка: В коде использован атрибут `style={{...}}` или переменная объекта.

Task 03
• Целевой файл: src/components/Footer/Footer.jsx
• Действие: Создайте подвал сайта, используя **внешний CSS-файл**.
1. Импортируйте файл `./Footer.css`.
2. Верните тег `footer` с классом `footer`.
3. Внутри добавьте любой текст копирайта.
• CSS-подсказка: (вставьте в src/components/Footer/Footer.css)
```css
.footer {
  background-color: #222;
  color: white;
  padding: 1rem;
  text-align: center;
  margin-top: auto;
}
```
• Проверка: Визуально: темный блок внизу страницы.

Task 04
• Целевой файл: src/pages/HomePage.jsx
• Действие: Реализуйте страницу для демонстрации работы с изображениями,.
1. **Импорт через JS:** Импортируйте изображение из `src/assets/programmer.jpg` в переменную. Создайте `div`, внутри него `h3` с подписью "Import" и тег `img`, использующий эту переменную в `src`.
2. **Папка Public:** Создайте второй `div`. Внутри него `h3` с подписью "Public" и тег `img`. В `src` укажите строковый путь к файлу из папки public (начинается с `/`).
• CSS-подсказка: Не требуется.
• Проверка: На странице отображаются две картинки (или иконки битых изображений, если файлы физически пусты, но пути в коде прописаны верно).

Task 05
• Целевой файл: src/App.jsx
• Действие: Соберите общую структуру приложения (Layout).
1. Импортируйте компоненты `Header`, `Footer` и `HomePage`.
2. Импортируйте `App.css`.
3. В `return` сформируйте структуру:
   - `<Header />`
   - Тег `<main>`, внутри которого рендерится `<HomePage />`.
   - `<Footer />`
• CSS-подсказка: Не требуется.
• Проверка: В браузере полная структура: Шапка -> Контент -> Подвал. Ошибок в консоли нет.
